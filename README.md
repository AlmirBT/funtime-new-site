# FunTime Website

Ultra-premium, large-scale frontend website for the FunTime Minecraft server. Built with React, TypeScript, TailwindCSS, and Framer Motion.

## Repository

GitHub: [funtime-new-site](https://github.com/yourusername/funtime-new-site)

### Initial Git Setup

If you're setting up this repository for the first time:

```bash
# Initialize git repository
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit: FunTime website"

# Add remote repository (replace with your GitHub username)
git remote add origin https://github.com/yourusername/funtime-new-site.git

# Push to GitHub
git branch -M main
git push -u origin main
```

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **TailwindCSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Routing
- **Lucide React** - Icons

## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn/pnpm

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```
